
Usage: node utils/streams.js [--help] [--action=(action) [--file=(filePath)]]

where (action) is one of:
    io, tr, trf, trsf

io          Pipes a file content to the process.stdout
tr          Transforms the process.stdin to upper case and pipes it to the process.stdout
trf         Transforms a file content to JSON format and pipes it to the process.stdout
trsf         Transforms a file content to JSON format and saves it

--help      Quick help on usage
aliases:    -h

--action    Action to be performed
aliases:    -a

--file      Path of a file to read
aliases:    -f
